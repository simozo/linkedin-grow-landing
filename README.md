# LinkedIn Grow Landing Page

Landing page for the LinkedIn Grow project, a tool designed to help users grow their presence and engagement on LinkedIn.

## Deployment

To deploy the project to Vercel, run the following command from your terminal:

```bash
git push vercel main --force
```

> [!NOTE]
> This command assumes you have the Vercel CLI configured and a remote named `vercel` pointing to your Vercel project.
